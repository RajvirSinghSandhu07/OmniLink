# 🌐 Omni Link: AI-Powered Humanitarian Logistics
Deployment Link: https://omni-link-final.vercel.app/

**Omni Link** is a high-performance, AI-driven operations platform designed for autonomous humanitarian aid coordination. Built for emergency response and large-scale volunteer management, it streamlines the bridge between donors, NGOs, and field volunteers using a dual-agent intelligence architecture.
![Omni Link Dashboard Mockup](https://images.unsplash.com/photo-1551288049-bebda4e38f71?auto=format&fit=crop&q=80&w=1200)
## 🚀 Vision
In the wake of urban crises or humanitarian needs, coordination is often the bottleneck. Omni Link eliminates the delay by using **Autonomous Dispatch Engines** to parse requests, evaluate logistical proximity, and deploy resources in real-time.
## 🛠️ Key Features
### 🧠 Autonomous Dispatch HUD
A command-line style interface where operators can "intervene" or suggest prompts. The system uses keyword-matching simulation (ready for LLM integration) to:
- **Parse Intake**: Identify resource type, quantity, and location.
- **Logistics Mapping**: Match requests with the nearest NGO hub.
- **Volunteer Deployment**: Assign specific units based on skills (Driver, First-Aid, etc.).
### 🗺️ Tactical Overlay & Network Graph
- **Bangalore Mission Map**: A high-tech visualizer showing glowing hotspots of need in real-time (e.g., Whitefield, Indiranagar).
- **Logistical Web**: An interactive network graph showing the active connections between Donor Clusters and NGO Partners.
### 📊 Mission Intelligence Core
- **Real-time Metrics**: Live tracking of Active Requests, Volunteers Online, and Response ETA.
- **Impact Analytics**: Recharts-powered sparklines and bar charts visualizing system reach and performance.
- **System Logs**: A transparent terminal tracking every internal "handshake" and API call (Maps Matrix, Node Connectivity, etc.).
### 📋 Operational Queue & Escalation
- **Interactive Lifecycle**: Move missions from "Incoming" to "Completed" with smooth Framer Motion animations.
- **Escalation Engine**: High-visibility alerts for critical medical or shelter needs requiring immediate supervisor intervention.
## 💻 Tech Stack
- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Visuals**: [Lucide React](https://lucide.dev/) (Icons), [Recharts](https://recharts.org/) (Data Viz)
- **UI Architecture**: Glassmorphism & "Digital Sovereignty" aesthetic
## 🏃 Getting Started
1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/omni-link.git
   cd omni-link
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Run the development server**
   ```bash
   npm run dev
   ```
4. **Visit the dashboard**
   Open [http://localhost:3000](http://localhost:3000) to view the Omni Link command center.
